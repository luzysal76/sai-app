### 1. 현재 서비스 구조의 강점과 약점

**강점**
- **명확한 포지셔닝**: "연애·관계 특화" AI 서비스로 집중도가 높음. 타로·사주 등 비과학적 도구를 제거한 것은 신뢰도 향상에 유리
- **3탭 구조의 단순함**: 홈/도구/일기 3탭은 인지 부하가 낮고 직관적
- **다층 접근 경로**: 홈(직통 AI 입력) + 도구탭(구조화 도구) + 일기탭(감정 기록)으로 사용 맥락을 넓게 커버
- **로컬 SPA 구현 완성도**: 서버 구조(server.js), OpenAI 챗봇, Claude Vision 캡처 분석까지 실질적 AI 기능 탑재

**약점**
- **사용자 여정이 단절됨**: 홈 AI 입력 → 챗봇 직통 연결 후, 도구탭과의 연결고리가 없음. 챗봇이 도구를 추천하거나 연결하는 플로우 부재
- **일기탭의 고립**: 감정 일기가 다른 기능(AI 상담, 도구)과 연동되지 않아 독립적으로만 존재
- **배포 미완성**: Firebase 미배포, ANTHROPIC_API_KEY 미설정 → 핵심 기능(캡처 분석)이 시뮬레이션 폴백 상태
- **재방문 유인 부족**: 챗봇 대화 히스토리, 진단 결과 저장, 관계 변화 추이 등 누적형 기능이 약함

### 2. 홈 AI 입력창 전략의 효과와 리스크

**효과**

- **연인 관계 한정 거절 로직**: "다른 관계 질문은 거절"이 사용자 경험을 경직시킬 수 있음 (부모·직장 관계도 관계 고민의 주요 영역)

### 3. 도구 16가지 핵심 vs 중복·불필요 판별

**핵심 도구 (Keep — 차별화 & 높은 수요)**

| 도구 | 이유 |
|------|------|
| 카톡 분석기 | 한국 시장 특화, 수요 명확 |
| 읽씹 분석 | 카톡 문화 특화, 감성적 니즈 |
| 답장 생성기 | 즉각적 가치, 높은 재사용 |
| 대화 캡처 분석 | Claude Vision 차별화 기능 |
| 사랑의 대화 (NVC) | 관계 심화, 교육적 가치 |
| 애착 유형 분석 | 자기이해 + 관계 이해, 체류시간↑ |
| 사랑의 언어 | 글로벌 검증된 프레임워크 |
| AI 챗봇 | 허브 역할, 모든 기능 연결점 |

**유지하되 고도화 필요**

| 도구 | 이유 |
|------|------|
| 관계 통장 | 개념 독창적이나 UI/기능 구체화 필요 |
| 관계 건강 체크 | AI 관계 진단(aidiag)과 중복 여지 → 통합 검토 |
| 성장 코치 | 범용성 높아 "연애 특화" 포지션과 충돌 가능 |

**중복·정리 대상**

| 도구 | 이유 |
|------|------|
| 관계 진단(diag) + AI 관계 진단(aidiag) | **사실상 동일 목적**, 하나로 통합 권장 |
| 표현 사전 | 사랑의 대화(NVC)와 기능 겹침 → 차별점 명확화 필요 |
| 대화 시나리오 | 답장 생성기와 유사, 역할 구분 불명확 |
| 대화 번역기 | "번역"이 단순 언어 번역이면 핵심 가치와 거리 있음 (감정 번역이면 Keep) |

> "카톡 하나로 연애 감정 분석 → 실전 답장까지" — 한국 연애 AI의 실용적 파트너

## 우선순위 2 🔴 API 비용 구조 설계 (단기, 생존 문제)

### 핵심 문제
수익 모델 없이 API 비용만 증가하면 서비스 지속 불가

### 비용 시뮬레이션 (gpt-4o-mini 기준)

| 사용자 수 | 1인/일 평균 토큰 | 월 토큰 | 월 비용 |
|-----------|----------------|---------|---------|
| 100명 | 5,000 | 15M | ~$0.9 |
| 1,000명 | 5,000 | 150M | ~$9 |
| 5,000명 | 5,000 | 750M | ~$45 |
| 10,000명 | 5,000 | 1.5B | ~$90 |

> gpt-4o-mini: $0.15/1M input + $0.60/1M output → 혼합 평균 ~$0.06/1M 토큰

**결론**: 1만 명 이하에서는 API 비용 자체는 관리 가능. 진짜 리스크는 **Claude Vision(캡처 분석) 비용**.

| claude-3-5-haiku | 이미지 1장 | ~1,000 토큰 = ~$0.001 |
|---|---|---|
| 1,000명 × 1장/일 | 월 30,000장 | ~$30/월 |

### 개선안: Freemium 게이트 설계
